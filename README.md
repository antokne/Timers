# Timers
Timer app for Homeworld Mobile

Allows you to create notifications for various events that occur in game.

Currently supports:
* Daily Market rest
* Daily Assingments
* Daily Liaison
* Daily Clan Assignments
* Community Thursday
* Weekly Market
* Weekly Assignments
* Weekend Event
* Iyatequa Liaison +XP
* Tanoch Liaison +XP
* Yaot Liaison +XP
* Amassari Liaison +XP
